Golden Grid System

An infinitely scaling grid for respon­sive web design.

Features

The main idea of Golden Grid System… fluid columns, elastic gutters, and a zoomable baseline grid.

Folding columns

Inspired by the DIN paper system. Splits the screen into 18 even columns. The leftmost and rightmost column as used as outer margins. The remaining 16 are perfect for creating layouts for large screens, and can be combined, or "folded", into 8 columns on medium-sized screens and 4 columns on small screens. Should cover anything from 240 to 2560 px. Allows easy use of the Golden Ratio at 8 columns (5/3 split) and 16 columns (10/6 split).

Basically infinite.

Elastic gutters

Elements in layouts that use fluid gutters often feel like they're stretching apart on large screens or being squeezed together on small screens. Using em-widths for the gutters gets you around this problem. They always stay in proportion to the content, as they should be. Accomplished by positioning elements in the middle of the gutter, and giving them a padding equal to half of the gutter's width.

Zoomable baseline grid

Type size presets based on the golden ratio. When font-size changes, all line-heights, margins, paddings, and borders scale in proportion, keeping the baseline grid intact. Meaning you can adjust the base font-size of your layout using media queries to, for example, make the text larger on huge monitors or tablets — both devices that tend to be further away from the eyes than usual — while keeping the baseline grid in harmony.

Delightful grid overlay script

A visual aid for both design and development. Works across pretty much all devices. Helps to make sure everything aligns up nicely with the grid. Appears with a pleasing animation on modern browsers. Give it a try by clicking or tapping the icon in the upper right corner of this page. It's fun.

Get the grid system

Golden Grid System is hosted on Github, and consists of three files.

ggs.html contains the required HTML, as well as a simple demonstration of the grid in use.

[ggs.css)(https://github.com/jonikorpi/Golden-Grid-System/blob/master/ggs.css) contains the CSS. Details of its contents are included in CSS comments throughout the file.

ggs.less contains the source of the CSS, written using the LESS CSS language. It allows you to change the base line-height (1.5) and works as a nice starting point for LESS users.

Get the Gridlet

A snippet for Javascript for overlaying the GGS grid on the page.

gridlet.js, when included in a page, adds an icon in the upper right corner that toggles the overlay. The overlay colours can be customized at the top of the file.

You can see a simple demonstration of the Gridlet in action in ggs.html.

Note that the Gridlet might not work as expected if the body element of your page has a set width and position: relative;, since the guides are appended inside body, but positioned in relation to the html element.
